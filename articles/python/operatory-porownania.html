Operatory porównania
śro 22 listopada 2017 python python / comparison / operator

Podczas tej lekcji będziemy uczyć się o Operatorach porównania w Pythonie. Umożliwiają nam one porównanie zmiennych oraz wyników otrzumując wartość logiczną (True lub False).

Jeżeli znasz podstawy matematyki to zagadnienie to, nie powinno być dla Ciebie obce.

Najpierw przyjrzyjmy się tabeli z operatorami porównania, a potem przejdziemy kilka przykładów:

Tabela Operatorów porównania¶

OperatorOpisPrzykład
== Jeżeli wartości dwóch argumentów są równe to warunek jest True. (a == b) nie jest True.
!= Jeżeli wartośći dwóch argumentów nie są równe to warunek jest True. (a != b) jest True
<> Jeżeli wartośći dwóch argumentów nie są równe to warunek jest True. (a <> b) jest True. Jest to podobne operatora != .
> Jeżeli wartość argumentu po lewej stronie jest większa niż po prawej to warunek jest True. (a > b) nie jest True.
< Jeżeli wartość argumentu po lewej stronie jest mniejsza niż po prawej to warunek jest True. (a < b) jest True.
>= Jeżeli wartość argumentu po lewej stronie jest większa lub równa niż po prawej to warunek jest True. (a >= b) nie jest True.
<= Jeżeli wartość argumentu po lewej stronie jest mniejsza lub równa niż po prawej to warunek jest True. (a <= b) jest True.
