Liczy - zaawansowane
śro 31 styczeń 2018 python numer / numbers / liczby

Podczas tej lekcji poznamy kilka więcej sposobów reprezentowania liczby w Pythonie

Hexadecimal (pl. System szesnastkowy)¶

Używając funkcji hex() mamy możliwość konwertowania liczb do systemu szesnastkowego:

In [1]:
hex(246)
Out[1]:
'0xf6'
In [2]:
hex(512)
Out[2]:
'0x200'

Binary (pl. system dwójkowy)¶

Używając funkcji bin() mamy możliwośc konwertowania liczb do systemu dwójkowego

In [3]:
bin(1234)
Out[3]:
'0b10011010010'
In [4]:
bin(128)
Out[4]:
'0b10000000'
In [6]:
bin(512)
Out[6]:
'0b1000000000'

pow()¶

Z dwoma argumentami funkcja ta jest równoznaczna z x^y, natomiast trzema argumentami (x^y)%z.

In [7]:
pow(2,4)
Out[7]:
16

abs¶

Wartość absolutna

In [8]:
abs(-3)
Out[8]:
3
In [9]:
abs(3)
Out[9]:
3

round (pl. zaokrąglić)¶

Zaokrąglenie liczby do danej dokładności cyframi dziesiętnymi (domyślnie 0 cyfr)

In [10]:
round(3)
Out[10]:
3.0
In [11]:
round(3.1415926535,2)
Out[11]:
3.14

Python posiada wbudowaną bibliotekę math, która jest także przydatna do obliczeń i operacji matematycznych. Tutaj znajduje się dokumentacjia.
